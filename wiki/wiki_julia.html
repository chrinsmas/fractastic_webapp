Julia Fractal

Pick an fixed complex number c. Now let's consider the map, $$z \rightarrow z^2 + c$$ If we start with z = c, what happens to z under repeated iteration?

The Julia Set (for a particular value c) is defined as the set of complex numbers for which z remains bounded under iteration of the map. That is, there exists a real number M such that $$\mid z\mid < M$$ for all steps in the iteration.
